## CAD paramétrico — Investigación y decisión

| Alternativa | Lenguaje | iOS | Kernel | Estado | Decisión |
|---|---|---|---|---|---|
| **SolveSpace** | C++ (GPL) | ✅ vía wrapper | NURBS + paramétrico + timeline | Kernel maduro, activo | **ELEGIDO** |
| **Manifold** | C++ (SDK Swift) | ✅ | Mesh (no NURBS) | SDK oficial | Complemento |
| **Cadova** | Swift DSL | ❌ solo macOS | Propietario | Pre-release | Descartado |
| **MiniCAD** | SwiftUI+SceneKit | ✅ iOS 16+ | SceneKit (no NURBS) | Legacy | Descartado |

**Decisión:** SolveSpace como kernel CAD. Razón: Único open-source con NURBS real, constraints paramétricas, timeline de operaciones. Shapr3D usa Parasolid ($299/año) — SolveSpace es la alternativa gratis. Implementación vía bridging header C → Swift.
