# Analisis Competitivo — AppForge Studio vs iPad 3D Market (Mayo 2026)

## Resumen
AppForge Studio es la unica app iOS nativa que combina CAD parametrico (OCCTSwift) + escultura + pintura 3D + animacion + exportacion a 5 formatos en una sola aplicacion. Ningun competidor directo ofrece esa combinacion.

## Competidores Identificados

### Shapr3D ($299/ano — Pro/Enterprise)
- CAD parametrico lider en iPad
- Sketch constraints, history-based modeling
- Export: STEP, IGES, STL, OBJ, USDZ
- NO tiene escultura, NO tiene pintura 3D, NO tiene animacion
- Target: ingenieros, disenadores industriales
- Pricing: gratuito para estudiantes; Pro ~$299/ano; Enterprise contacto

### Feather 3D ($14.99 unico pago)
- Dibujo 3D artistico (no CAD)
- Brushes 3D, liquify, animacion basica
- Export: OBJ, GLTF
- NO tiene CAD parametrico, NO tiene escultura, NO tiene export STEP
- Target: artistas 2D/3D
- 77 calificaciones, 4.09 estrellas

### Nomad Sculpt (~$35 desktop; iOS aparte)
- Escultura digital profesional
- Pinceles, deformers, pintura vertex
- Export: OBJ, STL, GLTF
- NO tiene CAD parametrico, NO tiene animacion
- Target: escultores 3D

### Forger ($9.99 unico)
- Escultura basica
- Pocas herramientas
- Sin CAD, sin animacion

## Ventajas Competitivas Clave
1. Unica app iOS con CAD parametrico + escultura en un solo flujo de trabajo
2. Precio potencial inferior a Shapr3D ($299/ano) con mas funcionalidades
3. Exportacion a 5 formatos incluyendo STEP para manufactura profesional
4. Modo oscuro nativo (Shapr3D no lo tiene en iPad)
5. Cache inteligente de modelos 3D para carga rapida

## Brechas a Cerrar para Competir Profesionalmente
1. Undo/Redo robusto con CADHistoryTree
2. Constraints parametricos completos (actual: sketch constraints basicos)
3. Mediciones precisas (dimensionamiento)
4. Material Editor con PBR textures
5. Exportacion FBX y Collada
6. TestFlight + AltStore para beta testing
7. UX pulida: onboarding, tooltips, gestos intuitivos

## Recomendacion Estrategica
- Pricing sugerido: $9.99/mes o $89.99/ano (vs Shapr3D $299/ano)
- Diferenciacion: "La unica app de diseno 3D todo-en-uno para iPad"
- Proximo lanzamiento: beta cerrada via TestFlight con 100 testers
